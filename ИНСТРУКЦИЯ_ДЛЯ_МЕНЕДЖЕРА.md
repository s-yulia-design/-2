## Два раздела в шапке

| **Населённые пункты** | Список всех НП, работа с каждым |
| **Сравнение** | Кто в приоритете — смотреть в первую очередь |

**Рейтинг:** зелёный 75+ перспективно · жёлтый 50–74 есть потенциал · красный ниже 50 слабый сигнал.

---

## Работа с новым населённым пунктом (по порядку)

**1. Создать** — главная → «+ Добавить населённый пункт» → название + район → «Создать».

**2. Справка** — «Найти в интернете» → сверить с **Росстатом (ЕМИСС)** и **ФИАС** по ссылкам → **«Подтвердить данные»**.

**3. Заявки** — внести улицу, дом, статус, источник. Галочка **«Подтверждена»** — если заявка проверена. Много заявок: «Скачать шаблон» → заполнить в Excel → «Импорт CSV».

**4. Конкуренты** — добавить провайдеров (название, технология, тариф, жалобы) → **«Подтвердить»**.

**5. Итог** — рейтинг, рекомендация, чего не хватает. Если рейтинг 75+ — передать на техобследование.

**6. Сравнение** — сопоставить с другими НП.

## Быстрые ответы

| Вопрос | Ответ |
|--------|--------|
| Где итог по НП? | Карточка НП → вкладка **Итог** |
| Почему низкий рейтинг? | Нет подтверждённых заявок, справки или конкурентов — смотрите блок «Не хватает данных» в **Итог** |
| Техника и Экономика? | Пока недоступны — заполнит техспециалист |
| Данные пропадут? | Нет, сохраняются на сервере |

**Демо для примера:** откройте любой НП из таблицы и нажмите «Найти в интернете» на вкладке Справка.
